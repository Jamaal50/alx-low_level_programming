# Argc, argv

Project done during **Full Stack Software Engineering Program** at **Holberton School**. It aims to learn about how to use arguments passed to a program in **C language**.

## Requirements
* Language: C
* Format: C89 Standard
* OS: Ubuntu 20.04 LTS
* Compiler: `gcc 4.8.4`
* Style Guideline: [Betty style](https://github.com/holbertonschool/Betty/wiki)

## Files
All of the following files are programs written in C:

| `0-whatsmyname` | Prints its name |
| `1-args.c` | Prints the number of arguments passed into it |
| `2-args.c` | Prints all arguments it recieves |
| `3-mul.c` | Multiplies two numbers |
| `4-add.c` | Adds positive numbers |
| `100-change.c` | Prints the minimum number of coins to make change for an amount of money |
