# Variadic Functions

Project done during **Full Stack Software Engineering Program** at **ALX Africa School**. It aims to learn about variadic functions in **C language**.

## Requirements
* Language: C
* Format: C89 Standard
* Compiler: `gcc 4.8.4`
* OS: Ubuntu 20.04 LTS
* Style Guideline: [Betty Style](https://github.com/holbertonschool/Betty/wiki)

## Files
All of the following files are programs written in C:

| Filename | Description |
| -------- | ----------- |
| `0-sum_them_all.c` | Returns the sum of all its parameters |
| `1-print_numbers.c` | Prints numbers |
| `2-print_strings.c` | Prints strings |
| `3-print_all.c` | Prints anything |
| `100-hello_holberton.asm` | Assembly that prints `Hello, Holberton` |
| `variadic_functions.h` | Header file containing all function prototypes |
