# Structures, typedef

Project done during **Full Stack Software Engineering Program** at **ALX Africa School**. It aims to learn about structures and `typedef` in **C language**.

## Requirements
* Language: C
* Format: C89 Standard
* Compiler: `gcc 4.8.4`
* OS: Ubuntu 20.04 LTS
* Style Guideline: [Betty Style](https://github.com/holbertonschool/Betty/wiki)

## Files
All of the following files are programs written in C:

| Filename | Description |
| -------- | ----------- |
| `dog.h` | Header file that contains a `struct dog` |
| `1-init_dog.c` | Initialize a variable of type `struct dog` |
| `2-print_dog.c` | Prints a `struct dog` |
| `4-new_dog.c` | Creates a new dog |
| `5-free_dog.c` | Frees a dog |
